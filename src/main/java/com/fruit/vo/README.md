# VO Module (View Object)

## 简介
`vo` (View Object) 包用于存放**后端接口返回给前端的数据对象**。

## 核心作用
1. **数据脱敏**：隐藏数据库 Entity 中的敏感字段（如密码、盐值、软删除标记）。
2. **格式化**：将数据库中存储的原始格式转化为前端友好的格式（如转换时间戳为字符串，将枚举值转化为中文描述）。
3. **数据聚合**：将多个 Entity 的数据组合成一个复杂的视图对象（如：商品详情 VO 可能包含商品信息 + 店铺信息 + 评价列表）。

## 区别
- **DTO**: 前端 -> 后端
- **Entity**: 后端 <-> 数据库
- **VO**: 后端 -> 前端
