# 水果生鲜电商系统 - 产品功能规划文档

## 一、项目概述

### 1.1 项目定位
B2B2C水果生鲜电商平台，单体架构设计，适用于毕业设计项目。

### 1.2 目标用户
- **消费者**：追求新鲜、品质的都市家庭用户
- **商家**：水果产地供应商、进口贸易商、本地农场
- **管理员**：平台运营人员

## 三、会员体系设计

### 3.1 会员等级

| 等级 | 所需积分 | 折扣率 | 权益 |
|------|----------|--------|------|
| 普通会员 | 0 | 100% | 基础服务、参与平台活动 |
| 银卡会员 | 500 | 98% | 98折、专属客服、生日礼包、优先发货 |
| 金卡会员 | 2000 | 95% | 95折、VIP客服、免运费券、专享活动 |
| 钻石会员 | 5000 | 92% | 92折、VIP客服、新品试用、积分翻倍 |

### 3.2 积分规则

| 积分来源 | 积分数量 | 说明 |
|----------|----------|------|
| 注册 | 100 | 新用户注册赠送 |
| 签到 | 10 | 每日签到，连续签到额外奖励 |
| 下单消费 | 1元=1积分 | 订单完成后发放 |
| 评价 | 5-20 | 评价质量越高积分越多 |
| 推荐注册 | 50 | 推荐新用户注册 |

## 五、板块导航设计

### 5.1 主导航栏

首页 | 新鲜水果 | 有机蔬菜 | 全球购 | 肉禽蛋品

### 5.2 首页功能块
```
┌─────────┬────────────────────────────┬─────────────┐
│ 商品分类 │ 轮播图 + 每日上新          │ 用户卡片    │

├─────────┴────────────────────────────┴─────────────┤
│ [猜你喜欢] [新鲜蔬菜] [时令水果]                    │
├─────────────────────────────────────────────────────┤
│ 商品网格展示区                                      │
└─────────────────────────────────────────────────────┘
```

### 5.3 推荐分类
1. **时令水果** - 苹果、橙子、芒果、草莓
2. **新鲜蔬菜** - 有机蔬菜、绿叶菜、根茎类
3. **肉禽蛋品** - 猪肉、鸡肉、鸡蛋、牛肉
4. **海鲜水产** - 鱼虾蟹、贝类、海带
5. **乳品冷饮** - 牛奶、酸奶、冰淇淋
6. **烘焙面点** - 面包、蛋糕、糕点
7. **酒水饮料** - 果汁、茶饮、矿泉水
8. **粮油干货** - 大米、食用油、干货

## 六、技术实现建议

### 6.1 前端技术栈
- Vue 3 + TypeScript
- Element Plus 组件库
- Pinia 状态管理
- Vue Router 路由

### 6.2 后端技术栈
- Spring Boot 2.7.18
- MyBatis Plus
- MySQL 8.0
- Redis 缓存
- JWT 认证

### 6.3 数据库设计原则
- 逻辑删除 (`deleted` 字段)
- 软删除优先，不直接删除数据
- 创建时间、更新时间自动填充
- 状态字段使用枚举管理

## 八、注意事项

1. **支付功能**：毕业设计项目，支付功能可模拟，暂不接入真实支付
2. **短信服务**：可使用阿里云短信或其他服务商，或暂时使用模拟验证码
3. **图片存储**：使用本地存储或对象存储服务
4. **安全考虑**：密码加密存储、敏感信息脱敏、XSS防护
